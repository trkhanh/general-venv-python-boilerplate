#  VENV + Python boilerplate

A simple boilerplate to production builds.

## Getting started

### Create your project folder.

```bash
# create new project
$ npx degit trkhanh/python-venv-boilerplate example-1
$ cd example-1

# install required venv
$ ./pre-install.ps1

# install dependency
$ ./install.ps1

# build the app
$ ./start.ps1
